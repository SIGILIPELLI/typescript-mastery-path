# TypeScript Mastery Path

A free, structured, module-wise TypeScript training program — entry level to master level, with runnable code in every module and a hands-on project at the end of each level.

**Live site:** https://sigilipelli.github.io/typescript-mastery-path/

## Contents

- **Level 1 · Entry** — language foundations & the type system
- **Level 2 · Intermediate** — idiomatic, testable TypeScript
- **Level 3 · Advanced** — APIs, databases, tooling
- **Level 4 · Master** — production systems & architecture

## Local development

```bash
python3 -m venv .venv
.venv/bin/pip install mkdocs-material
.venv/bin/python -m mkdocs serve
```
